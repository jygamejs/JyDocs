features:
  - title: Zero Dependencies
    details: Pure vanilla JavaScript ES modules. No bundlers, no frameworks, no bloat. Just import and play.
  - title: Canvas + DOM Hybrid
    details: Render game graphics on Canvas 2D while layering HTML UI elements on top — best of both worlds.
  - title: Scene Management
    details: Organize your game into scenes with lifecycle hooks — enter, exit, pause, resume, update, render.
  - title: Sprite & Group System
    details: Lightweight entity system with Sprites and Groups, including built-in collision detection.
  - title: Input Handling
    details: Keyboard (WASD, arrows, custom mappings) and touch input (swipe, tap) out of the box.
  - title: Math Utilities
    details: Vec2 and Rect classes for vectors, geometry, and spatial calculations in 2D space.
  - title: Asset Loaders
    details: Image and Font loaders with caching and batch loading for smooth game startup.
  - title: State Management
    details: Observable state container and localStorage wrapper for game state persistence.
